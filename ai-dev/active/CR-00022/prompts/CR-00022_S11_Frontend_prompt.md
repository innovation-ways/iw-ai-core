### 4. SSE row-update consumer

Replace the current `setTimeout(() => window.location.reload(), 800)` with row-level patching:

```javascript
es.addEventListener('row-update', function (ev) {
  const data = JSON.parse(ev.data);
  const row = document.getElementById('row-' + data.check_id);
  if (row) patchRowInPlace(row, data);
  else insertRow(data);  // domain group already exists
});
es.addEventListener('complete', function (ev) {
  // Just update the header pill — NO reload.
  refreshSummaryPill();
  closeProgressRow();
});
```

`patchRowInPlace` updates the status pill, severity pill, summary text, and `data-status` / `data-severity` attributes (so filter chips still work).
