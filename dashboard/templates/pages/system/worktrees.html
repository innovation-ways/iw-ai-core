Status guide

clean — no changes, safe to merge
dirty — tracked files modified; merge will fail
untracked — new files not staged; harmless but worth reviewing
orphaned — worktree on disk with no active batch item; safe to delete
no_path — path in DB does not exist on disk
error / timeout — git command failed

To remove an orphaned worktree: iw worktree-status to identify it, then delete via the item detail page or git worktree remove --force <path>
