Containers

What is this page?

Containers shows the health and resource usage of all Docker containers managed by IW AI Core — the per-worktree isolation containers and any sidecar services.

What can I do here?

  • Check container status and uptime
  • Inspect resource usage (CPU, memory)
  • Restart or remove a stalled container

Vocabulary

Worktree
A git worktree that runs inside an isolated Docker container per-agent-run.
Daemon
The background process that orchestrates containers and agent execution.
