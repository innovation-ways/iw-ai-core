Configuration

What is this page?

Configuration shows all runtime environment variables and feature flags loaded by the IW AI Core platform — both the global orch config and per-project overrides.

What can I do here?

  • Inspect the current environment and its effective values
  • Understand which values come from .env vs. project TOML vs. defaults
  • Validate that critical settings (DB host, polling interval, stall threshold) are as expected

Vocabulary

Daemon
The background orchestration process — reads config from .env and projects.toml.
Worktree
Per-run git worktree; each gets its own per-worktree DB when the project has ai-dev/iw-config/.
