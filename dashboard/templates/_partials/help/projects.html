Projects

What is this page?

The projects landing page lists every registered project in IW AI Core. Select a project to access its queue, history, docs, and all other workspaces.

What can I do here?

  • Browse all registered projects and their status
  • Create a new project by clicking the new-project button
  • Navigate into any project by clicking its card

Vocabulary

Project
A isolated workspace for a code repo, with its own queue, batches, docs, and RAG index.
Work item
A Feature, Incident, or Change Request tracked within a project.
