Test Coverage

What is this page?

Coverage shows aggregate code coverage data from the most recent test run — per-module, per-file breakdown with trend indicators.

What can I do here?

  • See which modules have the lowest coverage at a glance
  • Compare coverage trends across recent runs
  • Drill into a specific file to see uncovered lines

Vocabulary

Work item
Feature, Incident, or CR — coverage is measured per work-item or per full-suite run.
Step
One atomic test or quality gate within a work item.
