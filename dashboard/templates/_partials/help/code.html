Code (RAG)

What is this page?

Code is the RAG-backed code-understanding view — it indexes your repo, builds an architecture map, and lets you ask questions about any module with cited answers.

What can I do here?

  • Build or rebuild the semantic index for your codebase
  • Browse modules by name and explore their symbols
  • Ask free-text questions about any part of the codebase
  • View the auto-generated architecture diagram

Vocabulary

RAG / Code Understanding
Retrieval-Augmented Generation — indexes your code so the agent can answer questions with precise citations.
Fragment
A chunk of code or documentation extracted from a source file for embedding.
Work item
A Feature, Incident, or CR tracked in the system.
