Quality

What is this page?

Quality runs static analysis gates (lint, type check, formatter, security scans) against the project repo and surfaces pass/fail with clear error locations.

What can I do here?

  • Launch a quality gate run for any supported tool (ruff, mypy, etc.)
  • Stream live output as the run progresses
  • Inspect individual failures with file:line citations
  • Launch fix mode to auto-correct fixable errors

Vocabulary

Work item
A Feature, Incident, or CR — may carry a quality gate as part of definition.
Fix cycle
Auto-triggered re-run to correct quality failures.
