Worktrees

What is this page?

Worktrees shows every active git worktree the daemon has created for agent runs — with branch, last-commit status, and prune controls.

What can I do here?

  • Inspect git status of all active worktrees
  • Commit and prune stale worktrees that are no longer needed
  • See which batch each worktree is serving

Vocabulary

Worktree
An isolated git worktree created per-agent-run; disposable after merge.
Daemon
The orchestration daemon that creates worktrees and drives agent execution.
Batch
The daemon's scheduling unit — a named group of work items run together.
