Running Tasks

What is this page?

Running tasks is a live, cross-project view of every work item and batch currently being executed by the daemon — updated in real time via SSE.

What can I do here?

  • Watch the live progress of any in-flight work item
  • See which step is currently executing
  • Spot all running batches across all registered projects at once

Vocabulary

Work item
Feature, Incident, or CR — the unit the daemon executes.
Step
One atomic action within a work item (e.g., implement, test, doc, merge).
Batch
A group of work items launched together in one daemon pass.
Daemon
The background orchestration process that drives all execution.
