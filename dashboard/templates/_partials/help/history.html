History

What is this page?

History shows every batch that has been run for this project — past and present — in reverse chronological order.

What can I do here?

  • Inspect any historical batch to see which items ran, their outcomes, and any fix cycles
  • Filter by status (completed, failed, cancelled) or date range
  • Open the full execution report for any batch

Vocabulary

Batch
A group of work items launched together in one daemon pass.
Fix cycle
An additional agent run triggered automatically when a step fails, to attempt correction before surfacing the error.
Step
A single atomic action within a work item — e.g., implement feature, write tests, update docs.
