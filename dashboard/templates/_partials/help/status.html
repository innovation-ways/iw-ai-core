System Status

What is this page?

System status is a live health readout of the IW AI Core platform: daemon uptime, DB connectivity, worktree pool, and instance identity.

What can I do here?

  • Verify the daemon is polling and healthy
  • Check DB schema is up-to-date with head
  • Confirm instance identity matches expected (CR-00014)

Vocabulary

Daemon
The background process that polls PostgreSQL every 60 s and drives all agent execution.
Worktree
A git worktree created by the daemon for each agent run — isolated and disposable.
