Tests

What is this page?

Tests launches test suites per category (unit, integration, e2e) against the project repo, streams live output via SSE, and stores Allure reports for historical comparison.

What can I do here?

  • Select a test category and click Run to launch a suite
  • Watch live log output as tests execute
  • Browse historical Allure reports
  • Kill a stuck or runaway run

Vocabulary

Work item
A Feature, Incident, or CR — may carry its own test run as part of its definition.
Fix cycle
Auto-triggered retry after a test failure.
Step
One atomic test or action within a work item.
