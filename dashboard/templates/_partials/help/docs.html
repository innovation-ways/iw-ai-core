Docs Library

What is this page?

Docs is the versioned document catalogue for this project — everything generated, imported, or linked is stored here with full diff, stale detection, and HTML/PDF export.

What can I do here?

  • Browse all doc versions side-by-side with diffs
  • Run AI regeneration on stale or outdated documents
  • Export any doc as HTML or PDF
  • Validate links and check for lint warnings

Vocabulary

Stale doc
A document whose source code has changed since it was last rendered — a regeneration candidate.
Fragment
A section-level chunk of a document, rendered and diffed independently.
Work item
A Feature, Incident, or CR — the unit of work that may own or reference docs.
