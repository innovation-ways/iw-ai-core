Batches

What is this page?

The batches list shows every batch in this project — running, pending, held, and archived — updated in real time via SSE.

What can I do here?

  • View live status of all active batches
  • Pause, resume, or cancel a running batch
  • Archive a completed or cancelled batch to declutter the view
  • Create a new batch from the queue selection

Vocabulary

Batch
A named, executable group of work items — the daemon's atomic scheduling unit.
Step
One atomic unit of work inside a work item.
Fix cycle
Auto-triggered retry when a step fails; the daemon re-runs the agent to attempt correction.

Items released by an allow-on-overlap rule show an info pill — see Daemon Design for policy details.
