Batch Detail

What is this page?

Batch detail shows the full run record of a single batch: every work item inside it, their statuses, step-by-step logs, and any fix cycles applied.

What can I do here?

  • Drill into any individual work item within the batch
  • View logs and execution reports for each item
  • Restart merge or full-restart individual items from this screen

Vocabulary

Batch
A named group of work items run together in one daemon pass.
Work item
Feature, Incident, or CR — the unit the daemon schedules.
Fix cycle
Automatic retry after a step failure.

Items released by an allow-on-overlap rule show an info pill — see Daemon Design for policy details.
