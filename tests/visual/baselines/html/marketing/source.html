IW AI Core — Platform Product Overview

Type: Product Overview Version: v1.1 Status: Published Generated: 2026-05-09

Executive Summary

IW AI Core is an AI orchestration platform that drives LLM-agent development across multiple git worktrees. It schedules agents in background jobs, aggregates their output, and surfaces a web UI for code understanding, documentation, research, tests, and batch control.

For Engineering Managers: IW AI Core reduces the manual overhead of managing multiple AI agents, providing a single dashboard to monitor progress, review outputs, and ship merged code automatically.

The Problem We Solve

Running multiple AI agents across a codebase creates coordination overhead: keeping track of what each agent produced, ensuring consistency, merging changes, and catching regressions. Traditional approaches require either bespoke scripts or manual oversight on every batch.

Key Capabilities

  • Background batch scheduling — approve a work queue; agents process items in parallel git worktrees without blocking your terminal.
  • Web dashboard — queue management, live progress, RAG-backed code browsing, doc catalogue with diffs, and a unified jobs view.
  • Fix cycles — agents automatically re-run tests and attempt to resolve failures before reporting back.
  • Squash-merge to main — every successful batch collapses into a single clean commit.
  • CLI bridge — agents communicate results via iw step-done; no database files required on the agent side.

What Teams Use It For

  • Large-scale refactors that touch hundreds of files
  • Bulk documentation updates across multiple projects
  • Automated test coverage expansion
  • Research synthesis from multiple sources
